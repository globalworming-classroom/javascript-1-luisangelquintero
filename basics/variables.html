Variables

variables can be declared with 'const'

const x = 'hello';
console.log(x)
// -> 'hello'
// 'const' can't be changed though
x = 'change'
// -> Uncaught TypeError: Assignment to constant variable.

reassign variables by declaring them with 'let'

let y = 'hello';
console.log(y)
// -> 'hello'
y = 'change'
console.log(y)
// -> 'change'

use const wherever possible, there rarely is a need to reassign stuff, reassigning can be confusing for the reader
